# Design System: 연구실 케이지 관리 서비스

> **문서 버전**: v1.0 | **작성일**: 2026-01-18

---

## 1. 디자인 원칙

| 원칙 | 설명 | 참고 |
|------|------|------|
| **토스 스타일** | 현대적, 직관적, 부드러운 인터랙션 | 모던 UI |
| **구글시트 스타일** | 그리드 기반, 깔끔, 익숙한 표 형태 | 기능적 UI |
| **최소 클릭** | 불필요한 확인 팝업 최소화, 자동 저장 | UX 효율성 |

## 2. 색상 팔레트

### 2.1 Primary Colors

| 역할 | 이름 | HEX | 용도 |
|------|------|-----|------|
| Primary | Blue 500 | `#3B82F6` | 주요 버튼, 링크, 선택 상태 |
| Primary Light | Blue 100 | `#DBEAFE` | 배경 하이라이트 |
| Primary Dark | Blue 700 | `#1D4ED8` | 호버, 눌린 상태 |

### 2.2 Semantic Colors

| 역할 | 이름 | HEX | 용도 |
|------|------|-----|------|
| Success | Green 500 | `#10B981` | 성공 메시지, 완료 상태 |
| Warning | Amber 500 | `#F59E0B` | 경고, 충돌 알림 |
| Error | Red 500 | `#EF4444` | 오류, 삭제 |
| Info | Sky 500 | `#0EA5E9` | 정보 안내 |

### 2.3 Neutral Colors

| 역할 | 이름 | HEX | 용도 |
|------|------|-----|------|
| Gray 900 | 텍스트 | `#111827` | 본문 텍스트 |
| Gray 500 | 보조 텍스트 | `#6B7280` | 설명, 힌트 |
| Gray 200 | 테두리 | `#E5E7EB` | 카드, 입력 필드 테두리 |
| Gray 50 | 배경 | `#F9FAFB` | 페이지 배경 |
| White | 표면 | `#FFFFFF` | 카드, 모달 배경 |

### 2.4 교수별 케이지 컬러 (예시)

| 순서 | HEX | 용도 |
|------|-----|------|
| 1 | `#3B82F6` | 교수 A |
| 2 | `#10B981` | 교수 B |
| 3 | `#F59E0B` | 교수 C |
| 4 | `#8B5CF6` | 교수 D |
| 5 | `#EC4899` | 교수 E |

## 5. UI 컴포넌트

### 5.1 버튼

| 상태 | Primary | Secondary | Ghost |
|------|---------|-----------|-------|
| 기본 | Blue 500 배경, White 텍스트 | White 배경, Blue 500 테두리/텍스트 | 투명 배경, Blue 500 텍스트 |
| 호버 | Blue 700 배경 | Blue 50 배경 | Blue 50 배경 |
| 비활성 | Gray 300 배경, Gray 500 텍스트 | Gray 200 테두리, Gray 400 텍스트 | Gray 400 텍스트 |
| 크기 | 48px 높이 (터치 친화적) | 동일 | 동일 |

### 5.2 입력 필드

| 상태 | 스타일 |
|------|--------|
| 기본 | Gray 200 테두리, 12px radius |
| 포커스 | Blue 500 테두리, Blue 100 그림자 |
| 오류 | Red 500 테두리, Red 50 배경 |
| 비활성 | Gray 100 배경 |

### 5.3 케이지 셀 (그리드)

| 상태 | 스타일 |
|------|--------|
| 빈 케이지 | White 배경, Gray 200 점선 테두리 |
| 사용 중 | 교수 color_code 배경 (투명도 80%), 흰색 텍스트 |
| 호버 | 밝기 증가, 커서 포인터 |
| 선택됨 | Blue 500 테두리 (2px solid) |

### 5.4 토스트 (알림)

| 종류 | 스타일 |
|------|--------|
| 성공 | Green 배경, "✅ 메시지" |
| 경고 | Amber 배경, "⚠️ 메시지" |
| 오류 | Red 배경, "❌ 메시지" |
| 위치 | 화면 상단 중앙, 3초 후 자동 사라짐 |

### 5.5 모달/팝업

| 요소 | 스타일 |
|------|--------|
| 오버레이 | Black 50% 투명도 |
| 컨테이너 | White 배경, 16px radius, shadow-lg |
| 교수 선택 팝업 | 리스트 형태, 터치 영역 48px 높이 |

## 6. 반응형 브레이크포인트

| 이름 | 크기 | 대상 |
|------|------|------|
| Mobile | < 640px | 스마트폰 |
| Tablet | 640px - 1024px | 태블릿 |
| Desktop | > 1024px | PC |

### 6.1 그리드 반응형

| 디바이스 | 케이지 셀 크기 |
|----------|----------------|
| Desktop | 고정 (예: 60x60px) |
| Tablet | 화면 폭에 맞게 조정 |
| Mobile | 가로 스크롤 허용, 최소 터치 영역 44x44px 유지 |

---

## 7. 접근성 체크리스트

| 항목 | 기준 |
|------|------|
| 색상 대비 | WCAG AA (4.5:1) 이상 |
| 포커스 링 | 키보드 탐색 시 visible |
| 터치 영역 | 최소 44x44px |
| 텍스트 크기 | 최소 14px |
